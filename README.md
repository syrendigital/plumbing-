# Salt Lake City Plumbing Website

Resale-ready static website for a Salt Lake City plumbing company. Suggested resale value: **$500-$1,000**, depending on whether you include domain setup, copy customization, analytics, form integration, and launch support.

## What is included

- Responsive homepage for Salt Lake City plumbing searches
- Local SEO title, meta description, canonical URL, sitemap, robots file, and schema markup
- Service sections for emergency plumbing, drain cleaning, water heaters, leaks, fixtures, and sewer troubleshooting
- Privacy page and security header template
- Static lead request form that opens a prefilled email and does not store customer data
- Original local raster artwork, with no third-party stock image dependency

## Suggested pricing

For a clean handoff without custom setup, price around **$500-$650**.

For a stronger package with buyer branding, copy edits, form-provider setup, Google Business Profile alignment, Search Console submission, and launch support, price around **$750-$1,000**.

## Launch notes

Upload the folder to any static host such as Netlify, Cloudflare Pages, GitHub Pages, Vercel, or a conventional web host. Use HTTPS before public launch.

If the buyer wants online form storage instead of the mailto flow, choose a reputable form provider, enable spam protection, set short data retention, and update `privacy.html` to disclose where leads are stored.
